# Raycaster Project
The Bridge Between 2D Logic and 3D Illusion . Raycasting is a rendering technique that creates a pseudo-3D perspective from a 2D map by casting virtual "rays" from a viewer's position.

# features : 
 Mathematical Foundation<br>
 Real-time Interaction<br>
 Optimized Performance<br>
 
# Instalation (Linux , Ubuntu)
•To run the project on Linux, first make sure that the required build tools and libraries are installed.<br>
•Clone the project.<br>
•In the main project folder (where the Makefile is located) run:<br>
make run.<br>
• If only the executable file is to be built, use:<br>
make all.<br>
• To delete executable file:<br> 
make clean.<br>
